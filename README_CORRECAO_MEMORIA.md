# Correção - Memória

Alterações aplicadas:

- Padronização do tipo de produto `Memoria` para `Memória`.
- Correção automática de bancos antigos que já tinham `Memoria` salvo.
- Atualização de produtos e movimentações antigas que continham `Memoria` no nome.
- Importação de planilhas agora converte `Memoria`/`memoria` para `Memória`.
- Edição manual pela tela de Tipos de Produto também normaliza para `Memória`.
- Correções visuais simples na tela de Equipamentos.

Observação: isso não tinha relação direta com ABNT. Era um problema de valor padrão sem acento + banco já criado com dado legado.
